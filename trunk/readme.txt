== Description ==

Harika sözleri arşivleyip ve istediğiniz sayfalarda paylaşabilirsiniz...

== Installation ==

1. "/wp-content/plugins/" klasörünün içerisine "ahmeti_soz" klasörünü atınız.
1. 'Plugins' menüsünden eklentiyi aktif ediniz.
1. "<?php echo ahmeti_wp_guzel_sozler(); ?>" kodunu sözü nerede kullanmak isterseniz ekleyiniz.
